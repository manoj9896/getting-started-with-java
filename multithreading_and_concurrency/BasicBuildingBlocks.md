## Multitheading

- **Pros**
  - can enable more efficient use of CPU resources
  - allows things to run in parallel by utlilizing multiple CPU cores
  - Better resource utilization
  - Improved application responsiveness
  - Simplified program structure for certain tasks (e.g., I/O operations)

- **Cons**
  - Increased complexity in program design and debugging
  - Potential for concurrency issues
  - Overhead from context switching between threads

## Basic

- **Process** - Instance of a program (Has its own resources, at least one thread)
