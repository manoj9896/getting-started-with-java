# Nested Types

Types (classes, interfaces, enums, records) that are defined in other places than directly inside a package.
inside of other types or methods i.e. inside classes, interfaces, methods etc.

## Overview Nested Types

### classes

- static nested classes - `examples/example01`
- non-static nested classed (inner classes) - `examples/example02`

### Interfaces, records, enums

they are implicitly static

### Local Types

can be defined inside the methods or other scoped blocks - `examples/example03`

### Anonymous classes

defined directly on the spot - `examples/example04`
