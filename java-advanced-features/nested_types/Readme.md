# Nested Types
Types (classes, interfaces, enums, records) that are defined in other places than directly inside a package.
inside of other types or methods i.e. inside classes, interfaces, methods etc.

## Overview Nested Types
### classes
- static nested classes
- non-static nested classed (inner classes)

### Interfaces, records, enums
they are implicitly static

### Local Types
can be defined inside the methods or other scoped blocks

### Anonymous classes
defined directly on the spot
